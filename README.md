# Cinemastar

Local Movie Streaming Service

## Installation

Use `go get`:  

```shell
$ go get github.com/naoto/cinemastar-go
```

## Usage

Run Cinemastar  

```shell
$ ./cinemastar -f /path/to/movie/dir
```

any Option  

* `-f`: Select Movie Base Directory
* `-p`: Striming port

Access to `http://localhost:8080`
